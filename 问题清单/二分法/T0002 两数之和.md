# 两数之和

## 题目描述

给出一个有 n 个元素的数列 $a$ 和两个整数 $k$ 和 $s$，其中数列 $a$ 的元素是按照升序排列的。请你在数列中找出一个元素 $x$，使得 $x+k=s$。

## 输入格式

输入第一行一个整数 $n$ ，表示数列中的元素个数。

接下来一行输入 $n$ 个空格隔开的整数，表示输入的数列 $a$，保证是升序排列。

接下来一行输入两个整数 $k$ 和 $s$ 。

## 输出格式

如果能找到满足条件的数，输出 "Yes"，否则输出 "No"。

## 样例 #1

### 样例输入 #1

```
5
1 2 3 4 5
-1 4
```

### 样例输出 #1

```
Yes
```

## 提示

$1 \le n \le 10^5$

$-10^9 \le a \le 10^9$

$-10^9 \le k,s \le 10^9$
